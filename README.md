# Ebook: A Natureza em Minas Gerais

## 📒 Descrição
Este é um ebook que convida o leitor a conhecer um pouco mais sobre a riqueza natural de um dos estados brasileiros mais bonitos.

## 🤖 Tecnologias Utilizadas
    IA Generativa ChatGPT;
    IA Generativa BRI.IA para criar a capa do e-book;
    Ferramenta Microsoft Word para edição.

## 🧐 Processo de Criação

 1) Com comandos no prompt, solicitei ao ChatGPT exemplos ebooks sobre temas variados
 2) Fiz a escolha do tema para o ebook
 3) Pesquisei algumas imagens no google sobre o tema para assim, ter inspiração de como seria incluido os conteudos
 4) Escrevi no prompt de comando do ChatGPT, algumas frases chaves que serviram de base para criar os conteudos
 5) Com comandos no prompt, detelhei ao maximo as frases chaves 
 6) Solicitei ao ChatGPT exemplos de como seria a estrutura de um ebook
 7) Realizei a montagem do ebook no Microsoft Word
